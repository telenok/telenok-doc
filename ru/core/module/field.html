Модуль "Поле"

Модуль "Поле" содержит инструменты для манипуляции полями типов.

При добавлении поля в тип в таблицу типа добавляется соответсвующий столбец, если поле простое вроде Целое, Строка, или создается новая pivot-таблица для полей типа связи "многие-ко-многим".

На рисунке показан модуль "Поле". 

1. Модуль "Поле" в Группе модулей "Контент" и Главного модуля "Объекты"
2. Дерево типов и папок. Вы можете выбрать тип и увидеть в табе справа в списке (3) поля, из которых этот тип состоит.
3. Список полей типа.
2. Столбец кодов полей.

Для создания нового поля объекта нажмите на кнопку, представленную на следующем изображении.

Создание нового поля объекта. 

1. Кнопка "Создать"

После нажатия на кнопку "Создать" откроется новая вкладка, как показано на следующем изображении.

Элементы вкладки "Главное" модуля "Поле". 

1. Поле "Номер".
2. Поле "Название".
3. Поле "Название в списке". Задается во множественном числе.
4. Поле "Код". Значение должно быть уникальным для полей данного типа. В зависимости от поля (целое, строка, отношение с другими типами объектов) этим названием
может стать столбец в таблице базы данных.
5. Поле "Отношение к типу". Задается тип объекта, к которому относится данное поле.
6. Поле "Тип поля". Можно выбрать только при создании Поля. Изменить невозможно через административный интерфейс. Задается тип поля. Возможные варианты:
- Целое
- Целое положительное
- Строка
- Мультистрочный текст
- Массив
- Отношение один-к-одному
- Отношение один-ко-многим
- Отношение многие-ко-многим
- Дерево (иерархия)
- Полиморфное отношение один-к-одному
- Полиморфное отношение один-ко-многим
- Полиморфное отношение многие-ко-многим
- Связанные файлы-объекты
- Загрузка файла
- Выбор одного из многих элементов
- Выбор нескольких элементов из многих элементов
- Время ЧЧ:ММ:СЕК, например, 18:09:20
- Дата+время ГОД-МЕС-ЧИСЛО ЧЧ:ММ:СЕК, например, 2014-11-20 08:08:19
- Диапазон времени, например, 08:08:19 - 09:21:55
- Диапазон дата+время, например, 2014-11-20 08:08:19 - 2015-12-01 18:58:39
7. 8. 9. Опциональные параметры поля

Для настройки видимости поля и некоторых опций отображения в форме перейдите на вкладку "Дополнительное"

Элементы вкладки "Дополнительное" модуля "Поле". 

1. Вкладка "Дополнительное".
2. Поле выбора вкладки в элементе объекта связанного типа.
3. Поле "Доступ". Вы можете задать доступ пользователей и объектов к данному элементу.
4. Выбор "Отображение при создании". Если выбрано значение "Нет", то при создании элемента типа это поле не будет доступно для заполнения.
5. Выбор "Отображение при редактировании". Если выбрано значение "Нет", то при редактировании элемента типа это поле не будет доступно для заполнения.
6. Выбор "Разрешена сортировка" в списке элементов данного типа по этому полю.
7. Выбор "Разрешен поиск" в списке элементов данного типа по этому полю.
8. Выбор "Показывать в списке" в виде столбца.
9. Выбор "Показывать в форме" это поле.
10. Поле "Описание". Появится в виде круглого синего значка и всплывающей подсказки при наведении на значок рядом с полем в форме элемента типа.
11. Поле "CSS класс" элемента в форме.
12. Поле "CSS класс иконки" в форме.
13. Поле "Порядок отображения" в форме на вкладке.
14. Служебные поля о времени и авторе создания, редактирования и блокировки записи.
