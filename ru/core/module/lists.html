Модуль "Список"

Модуль "Список" содержит записи всех типов данных, представленных в системе. В этом списке Вы имеете возможность найти, просмотреть и отредактировать любую запись любого типа.

Необходимо иметь в виду, что тип и элементы списков можно ограничить в соответствии с правами доступа для конкретного пользователя или процесса.

Модуль "Список" находится в Группе модулей "Объекты".

На рисунке показан модуль "Список". 

1. Модуль "Список" в Группе модулей "Контент" и Главного модуля "Объекты"
2. Дерево папок. Содержит все зарегистрированные типа данных
3. Вкладка со списком элементов выбранного типа данных в дереве (2). 
Закрыть вкладку Вы можете нажатием на красное перекрестие справа от названия или средней клавишей мыши по названию вкладки.
4. Кнопка "Редактировать запись"
5. Иконка активности элемента
6. Иконка блокировки элемента. Если элемент открыт кем-то, кроме Вас, то замок на иконке станет закрытым. 
В таком случае Вы не сможете сохранить запись прежде, чем её прекратит редактировать или просматривать другой пользователь.
Если пользователь прекратит редактирование или просмотр записи, например, закрыв вкладку, то запись станет доступна для вас в течение 5-10 минут после этого.
7. Кнопка "Удалить запись"

Многие модули содержат общие элементы управления списками, представленные на следующем изображении.

Общие элементы управления некоторых модулей. 

1. Кнопка поиска по дереву
2. Кнопка обновления дерева
3. Кнопка сокрытия дерева. Для показа дерева кликните на соответствующем модуле в меню слева
4. Кнопка "Создать запись"
5. Кнопка "Обновить список"
6. Кнопка для выбора действия над указанными в (7) записями
7. Кнопки массового выбора записей
8. Кнопка "Фильтр списка"
9. Поле поиска записей по части названия или номеру

Многие модули содержат общие элементы поиска в списках, представленные на следующем изображении.

Общие элементы поиска некоторых модулей. 

1. Поле ввода части названия для поиска в дереве
2. Кнопка "Найти"
3. Кнопка "Сбросить" для возврата дерева в исходный вид
4. Кнопка "Фильтровать записи". При нажатии на эту кнопку открывается форма с динамическими полями фильтрации (7), которые можно добавить или скрыть.
5. Кнопка "Найти"
6. Кнопка "Очистить" фильтр для возврата списка в исходное состояние
7. Кнопки выбора полей фильтрации
8. Поле глобального поиска по всем записям приложения
