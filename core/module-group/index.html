Группа модулей

Группа модулей - это четыре иконки в верхнем левом углу, каждая из которых соответствует группе модулей.

При нажатии на одну группу из четырех (под цифрой 1) появляются Главные модули (под цифрой 2). При нажатии на Главном модуле появляются Модули (цифра 3)

На изображении показаны Группа модулей (1), Главный модуль (2), Модуль (3) и кнопка минимизации (4) Группы модулей.

В приложении всего 4 Группы модулей:

- Контент
- Пользователь
- Веб
- Настройки
